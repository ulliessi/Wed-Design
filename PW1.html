Question a

HTML5 became a standard on October 28, 2014. The main reasons for developing this new standard were to make the language better with support for the latest multimedia and to ensure it’s easy to read by people and consistently understood by computers and devices.

Question b

The main difference between W3C HTML5 and WHATWG HTML Living Standard is that the W3C version uses version numbers and is considered a snapshot of the “Living Standard” from WHATWG. The WHATWG version is constantly updated and doesn’t have version numbers.

Question c

The minimal code for an HTML page looks like this:

<!DOCTYPE html>

<html>

<head>

<title>Title of the document</title>

</head>

<body>

<h1>This is a heading</h1>

<p>This is a paragraph</p>

</body>

</html>

Question d

Semantic HTML describes the content and structure of a document, while presentational HTML is used to determine the look of the document. Presentational markup is considered outdated because it goes against the principle of separation of concerns, according to which HTML should be used for structure, and CSS for presentation.

Question e

You can check browser support for a specific tag or attribute using resources like “Can I use” or JavaScript libraries like Modernizr.

Question f

The audio and video formats officially supported by HTML standard are MP3, WAV, and Ogg for audio, and MP4, WebM, and Ogg for video. If you use another format, it may not be supported by all browsers, which means the audio or video will not play correctly.
